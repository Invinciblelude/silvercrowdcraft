‚úÖ Build Sequence Checklist:

  1. Foundation: Slab poured and cured (7 days)
  2. Bottom Plates: Bolt to foundation with anchor bolts
  3. Exterior Walls: Frame flat, tip up, brace temporarily
  4. Interior Load Walls: Frame and install load-bearing walls
  5. Double Top Plates: Nail two layers on top of all walls
  6. Support Beams: Install LVL beams that span across walls
  7. Ridge Beam: Install peak beam (need scaffolding/lift)
  8. Rafters: Install all roof rafters 24" on center
  9. Roof Sheathing: Nail OSB/plywood to rafters
  10. INSPECTION: Call for framing inspection before continuing!
