Process and controls

  1. Underwrite: ARV by zip and spec; max-offer from residual method.
  2. Pre-construct: ALTA/topo, utility will-serve, fee worksheet, test-fit/site plan.
  3. Build: milestone schedule; QC photo gates; retainage on GC draws.
  4. Sell: staged listings, DOM targets by submarket, weekly pricing reviews.
  5. Report: weekly updates + monthly KPI (budget vs actual, cash, risks).

Investor FAQ

How do I invest?

We raise via a compliant pathway. If using Regulation Crowdfunding, checkout occurs on a FINRA-registered portal. If using Reg D 506(c), only accredited investors may participate with accreditation verification. Your “Invest” click will route accordingly.

What returns do you target?

We target per-project margins of 12–18% when acquired within our buy box and executed to spec. No returns are guaranteed. We publish ranges and risks and report budget vs actual monthly.

What is the liquidity and timeline?

Our lot-build turns target 120–150 days from notice-to-proceed to listing/sale. Liquidity depends on the security type and offering; some securities restrict transfer for 1 year.

How do you manage risk?

Finished-lot basis guardrails (≤ 15–20% of ARV), 10% contingency, GC retainage, milestone draws, two-agent exit comps, and weekly site cadence. We proactively re-price if DOM deviates by submarket.
