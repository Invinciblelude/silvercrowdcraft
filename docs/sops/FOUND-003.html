FOUND-003: Stem Wall & Foundation Wall

Trade Foundation
Duration 32 hours
Crew Size 3 people

üìã Prerequisites

  • Footings cured (7 days minimum)
  • Footing inspection passed

üß∞ Materials Needed

  • Concrete blocks (CMU) or concrete for poured wall
  • Rebar (vertical and horizontal per plan)
  • Mortar or concrete
  • Anchor bolts (1/2 inch, per plan)
  • Form lumber if poured wall (2x10 or 2x12)
  • Form ties and walers

üîß Tools Required

  • Block saw
  • Trowel
  • Level
  • String line
  • Rebar cutter
  • Drill for anchor bolts
Back support belt (blocks are heavy)
Gloves
Steel-toe boots
Eye protection

üìù Step-by-Step Instructions

  1. Snap chalk line on footing for stem wall alignment
  2. If CMU: Lay first course on mortar bed, check level and plumb
  3. Continue courses, installing horizontal rebar in bond beams
  4. Fill block cores with grout and vertical rebar per plan
  5. Install anchor bolts in top course (typically 6ft spacing, 7 inches embedded)
  6. If poured: Erect forms, install rebar, pour and vibrate concrete
  7. Cure for 7 days
  8. Backfill against wall only after curing (use gravel for drainage)

‚úÖ Quality Control Checks

Wall is plumb and level
Height matches plan
Anchor bolts are vertical and at correct spacing
Rebar is fully grouted (no voids)
Top of wall is level for sill plate

  • Foundation wall inspection before backfill
