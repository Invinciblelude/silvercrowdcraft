Duration 16 hours
Crew Size 2 people

üîß Tools Required

  • Adjustable wrench

üìù Step-by-Step Instructions

  1. Install toilets: set wax ring on flange, set toilet, bolt down, connect supply line
  2. Install sinks: mount sink, connect faucet, attach P-trap to drain, connect supply lines
  3. Install shower fixtures: attach trim plate, install shower head and valve handle
  4. Test all fixtures for leaks (run water for 5 minutes)
  5. Caulk around base of toilets and sinks (not back of toilet)
  6. Install shower doors or curtain rods
  7. Ensure all shut-off valves are accessible and working
