FRAME-004: Roof Framing

Trade Framing
Duration 48 hours
Crew Size 4 people

üìã Prerequisites

  • Wall framing complete and inspected

üß∞ Materials Needed

  • Rafters or trusses (per plan)
  • Ridge board (if stick-framed)
  • Ceiling joists (if not using trusses)
  • Hurricane ties for rafter connections
  • Fascia board
  • Soffit material
  • Frieze blocking
  • 3/2 inch ring shank nails

üîß Tools Required

  • Circular saw
  • Framing nailer
  • Hammer
  • Level
  • Chalk line
  • Speed square
  • Ladder or scaffolding
Fall protection (harness and lifeline)
Hard hat
Safety glasses
Non-slip boots
Tool belt

üìù Step-by-Step Instructions

  1. If trusses: Crane or multiple crew lift trusses into place, space per plan (typically 24 inch OC)
  2. If stick-framed: Install ceiling joists first, then ridge board, then cut and install rafters
  3. Attach rafters to top plate with hurricane ties (critical for wind resistance)
  4. Install collar ties or rafter ties per code (typically every 4ft)
  5. Check roof plane is straight (no dips or humps)
  6. Install fascia board at rafter tails
  7. Install frieze blocking between rafters at top plate
  8. Install soffit and vents for attic ventilation

‚úÖ Quality Control Checks

Roof pitch matches plan
All rafters/trusses are plumb and spaced correctly
Hurricane ties installed at every connection
Ridge board is straight
Overhangs are consistent
Soffit vents provide adequate attic ventilation (1:150 ratio)
