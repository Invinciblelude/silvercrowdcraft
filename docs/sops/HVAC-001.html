üìù Step-by-Step Instructions

  1. Install HVAC unit in attic, garage, or exterior pad (per plan)
  2. Run main supply trunk line from unit to central location
  3. Branch flex ducts to each room per plan (sized for CFM requirement)
  4. Install supply registers in ceiling or floor per plan
  5. Install return grille in central location (typically hallway)
  6. Run refrigerant lines from condenser to air handler (if split system)
  7. Run condensate drain line to exterior or drain
  8. Seal all duct joints with mastic (not just tape)
  9. Insulate ducts in unconditioned spaces
