ROOF-003: Shingle Installation

Trade Roofing
Duration 32 hours
Crew Size 3 people

üìã Prerequisites

  • Underlayment and flashing complete
  • Dry weather forecast

üß∞ Materials Needed

  • Asphalt shingles (30-year architectural grade typical)
  • Starter strip shingles
  • Ridge cap shingles
  • Roofing nails (1-1/4 inch)
  • Roofing cement for penetrations

üîß Tools Required

  • Roofing nailer
  • Utility knife
  • Chalk line
  • Hammer
  • Harness and lifeline
Fall protection (100% tie-off)
Hard hat
Non-slip boots
Knee pads
Safety glasses

üìù Step-by-Step Instructions

  1. Install starter strip along eave (adhesive strip facing up)
  2. Snap horizontal chalk lines for shingle courses (every 5 inches typically)
  3. Install first course of shingles along eave, overhang drip edge by 1/2 inch
  4. Nail each shingle with 4-6 nails per manufacturer instructions
  5. Offset each course by 6 inches (half-tab) for pattern
  6. Work up roof, following chalk lines
  7. Cut and weave step flashing at wall intersections
  8. At ridge, install ridge cap shingles with overlap, nail on downslope side
  9. Seal all penetrations (vents, pipes) with roofing cement and boot flashings

All shingles are aligned and nailed correctly
Shingle exposure is consistent (typically 5 inches)
Ridge caps are sealed and nailed properly
All penetrations are sealed and flashed
No exposed nail heads
