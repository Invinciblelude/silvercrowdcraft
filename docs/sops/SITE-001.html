üìù Step-by-Step Instructions

  1. Review site plan and locate property corners using survey markers
  2. Set up transit at benchmark elevation
  3. Drive corner stakes at building footprint corners per plan dimensions
  4. Run string lines between corner stakes to establish building outline
  5. Mark foundation excavation lines 2ft beyond building footprint
  6. Mark utility trenches (water, sewer, electric) per civil plans
  7. Spray paint all excavation boundaries with bright color
  8. Photograph staked layout for record
