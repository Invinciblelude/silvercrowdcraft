## üèóÔ∏è **Phase 4: Full Project - Your 728 Cordant House (Weeks 7-10)**

Now you model your ENTIRE house, step by step.

### Week 7: Foundation & Walls
**Goal:** Model all walls accurately

**Process:**
1. Import floor plan as reference image
2. Set scale correctly (1/4" = 1'-0")
3. Create wall types:
   - Exterior: 2x6 stud wall (6" thick)
   - Interior: 2x4 stud wall (4.5" thick)
4. Trace EVERY wall from blueprint
5. Set wall heights (8'-0" standard, vaulted for Great Room)

**Challenges You'll Face:**
- Angled walls (north wing at 45¬∞)
  - Solution: Use "Rotate" and "Align" tools
- Complex room shapes
  - Solution: Draw one wall at a time, trim as needed

### Week 8: Doors, Windows, Roof
**Goal:** Complete openings and roof structure

**Process:**
1. Add all doors (reference door schedule from blueprint)
2. Add all windows (reference window schedule)
3. Create roof:
   - Use "Roof by Footprint" tool
   - Set slope to 4:12
   - Add structural framing (beams per Sheet S-3)

### Week 9: Details & Refinement
**Goal:** Add finishing touches

**Process:**
1. Add foundation (slab on grade)
2. Add materials:
   - Stucco exterior
   - Drywall interior
3. Create proper views:
   - Floor plan
   - All elevations
   - 3D views

### Week 10: Rendering & Output
**Goal:** Create professional presentation

**Process:**
1. Set up lighting
2. Render 5-10 images:
   - Exterior (front, rear, side)
   - Interior (Great Room, Master Bedroom, Kitchen)
3. Export files:
   - .rvt (Revit native)
   - .dwg (AutoCAD)
   - .pdf (for viewing)
4. Create walkthrough video
